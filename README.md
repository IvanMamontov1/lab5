# lab5
Проект представляет собой учебное приложение для управления задачами.
Цель проекта — смоделировать процесс разработки по Scrum.
Работа организована с использованием GitHub Issues.
В рамках лабораторной работы сформирован Product Backlog.
Проведено планирование одного спринта и отслежен его прогресс.
## Sprint Review

В рамках первого спринта были выполнены задачи по регистрации и авторизации пользователя.
Реализован базовый функционал создания задач.
Часть задач, связанных с управлением статусами, осталась незавершённой и будет перенесена в следующий спринт.

## Sprint Retrospective

### Что прошло хорошо
Задачи были чётко сформулированы, приоритеты определены заранее.
Использование GitHub Issues упростило отслеживание прогресса.

### Что пошло не так
Некоторые задачи оказались сложнее, чем ожидалось, из-за недооценки сложности.

### Что можно улучшить
В следующем спринте стоит более точно оценивать story points и уменьшать объём задач.
